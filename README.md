# Telegram News Bot

Telegram-бот, який збирає новини з RSS-каналів (BBC, Голос Америки, RFI) і надсилає їх користувачам відповідно до обраних тем.

## Функціонал

- Вибір тем новин через інлайн-клавіатуру (`/start`)
- Підтримка українських RSS-каналів
- Підтримка RFI (FR) з автоматичним перекладом на українську
- Фільтрація новин за ключовими словами
- Перевірка, щоб одна й та ж новина не надсилалася повторно
- Розсилка кожні 30 хвилин
